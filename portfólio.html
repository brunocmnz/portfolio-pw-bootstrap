Sistema de gerenciamento de supermercado em Java

Projeto desenvolvido durante a disciplina de Programação de Computadores 2, com o intuito de praticar conceitos como Programação Orientada a Objetos.

Este trabalho tinha por objetivo o desenvolvimento de módulos administrador e atendente, sendo que, para cadastrar compras é preciso estar no módulo atendente, e já para alterar dados é preciso estar no módulo administrador.

Problema do Jantar dos Filósofos em Java

Projeto desenvolvido durante a disciplina de Sistemas Operacionais, buscando utilizar programação concorrente, gerenciando as threads, com uma visão gráfica da operação, simulando o problema do jantar dos filósofos.

O problema simula a concorrências de recursos entre threads, em que cada filósofo precisa de dois talheres que estão ao seu lado, mas que podem estar sendo usados pelo filósofo adjacente.
